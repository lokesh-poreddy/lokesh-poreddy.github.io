# Poreddy Lokesh Reddy - 3D Portfolio

A stunning interactive 3D portfolio built with **Next.js**, **React Three Fiber**, and **Tailwind CSS**. Features smooth animations, dark/light theme support, and immersive visual effects.

## 🚀 Live Portfolio
Visit your portfolio at: **https://lokesh-poreddy.github.io/**

## ✨ Features
- ✅ 3D spatial background with mouse tracking
- ✅ Animated hero section with gradient text
- ✅ Interactive achievements with 3D rotating badges
- ✅ Fun 8-bit animation scene (Naruto vs Sasuke clash)
- ✅ Dark/Light theme toggle
- ✅ Fully responsive design
- ✅ TypeScript & modern React patterns

## 📦 Tech Stack
- **Next.js 15** - React framework
- **React Three Fiber** - 3D graphics
- **Tailwind CSS** - Styling
- **Radix UI** - Component library
- **TypeScript** - Type safety

## 🔧 Development
```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Deploy to GitHub Pages
pnpm build  # Static export to docs/
git add docs
git commit -m "Update portfolio"
git push
```
